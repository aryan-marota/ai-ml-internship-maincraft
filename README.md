# AI & Machine Learning Internship – Maincraft Technologies

This repository contains all tasks completed during my **6-week AI & Machine Learning Internship**
at **Maincraft Technologies**.

---

## 📌 Tasks

### Task 1: House Price Prediction using Linear Regression
- Built and evaluated a Linear Regression model
- Performed EDA on the California Housing Dataset
- Evaluated using MAE, RMSE, and R²

### Task 2: Model Comparison for House Price Prediction
- Trained multiple regression models
- Compared performance using MAE, RMSE, and R²
- Selected the best-performing model based on evaluation metrics

### Task 3: Model Validation and Hyperparameter Tuning
- Detected and analyzed overfitting
- Used cross-validation for reliable evaluation
- Applied GridSearchCV for hyperparameter tuning
- Selected final model based on generalization performance

### Task 4: Classification Model Evaluation
- Built a binary classification model
- Evaluated using precision, recall, F1-score, and ROC-AUC
- Handled class imbalance using class weights
- Compared Logistic Regression with Decision Tree
